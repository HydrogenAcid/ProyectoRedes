# Proyecto de Redes

Repositorio para trabajar el proyecto individual y el proyecto grupal de la materia de Redes. Favor de subir los scrips con los metodos que hemos expuesto en clase

## Estructura del proyecto

```text
Proyecto de redes/
|-- Bases de datos/      # Archivos de datos utilizados por los scripts
|-- Braulio/             # Scripts y archivos de Braulio
|-- Jardi/               # Scripts y archivos de Jardi
|-- Rich/                # Scripts y archivos de Rich
|-- requirements.txt     # Dependencias de Python
`-- readME.md            # Documentacion del proyecto
```

Cada integrante debe subir sus scripts dentro de la carpeta correspondiente a su nombre.

## Flujo de trabajo con Git

Antes de empezar a trabajar, revisa el estado del repositorio:

```bash
git status
```

Para guardar y subir cambios:

```bash
git add .
git commit -m "Describe el cambio realizado"
git push
```

## Recomendaciones

- No subir el entorno virtual `.venv/`.
- No subir archivos temporales ni caches de Python.
- Mantener actualizado `requirements.txt` cuando se agreguen nuevas librerias.
